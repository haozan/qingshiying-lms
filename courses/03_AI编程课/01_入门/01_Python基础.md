# Python基础

Python是AI开发的首选语言。本节课将学习Python的基础语法。

## 变量与数据类型

```python
# 数字
age = 25
price = 19.99

# 字符串
name = "青狮营"

# 列表
fruits = ["苹果", "香蕉", "橙子"]

# 字典
user = {
    "name": "张三",
    "age": 25
}
```

## 条件语句

```python
if age >= 18:
    print("成年人")
else:
    print("未成年人")
```

## 循环

```python
for fruit in fruits:
    print(fruit)
```
