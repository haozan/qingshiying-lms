# 图片理解

## 场景描述

本节将演示如何使用阿里云通义千问 qwen-vl-plus 视觉模型，批量提取带水印试题图片中的文字内容，实现内容数字化变现。

## 实战案例

### 业务场景
- 原始素材：970 道带水印的试题图片
- 处理目标：提取图片中的题目、选项、答案等文字内容
- 变现方式：整理成题库、制作成电子书、开发在线练习系统

### 技术方案
- 使用模型：阿里云通义千问 qwen-vl-plus
- 处理能力：支持图片中的文字识别、理解图片内容、提取结构化信息
- API 调用：通过 OpenAI 兼容接口调用

### 实现步骤

1. **准备工作**
   - 获取阿里云 API Key
   - 准备试题图片素材
   - 设计提示词模板

2. **调用视觉模型**
   - 上传图片或传递图片 URL
   - 发送识别请求
   - 接收结构化结果

3. **批量处理**
   - 遍历 970 张图片
   - 自动提取内容
   - 保存为结构化数据（JSON/Excel）

4. **内容变现**
   - 清洗和校对提取的内容
   - 整理成题库产品
   - 发布到知识付费平台

## 关键代码示例

```python
import openai

# 配置阿里云通义千问 API
client = openai.OpenAI(
    api_key="your-dashscope-api-key",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
)

# 调用视觉模型提取图片内容
response = client.chat.completions.create(
    model="qwen-vl-plus",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "请提取这张图片中的试题内容，包括题目、选项和答案"},
                {"type": "image_url", "image_url": {"url": "图片URL或base64"}}
            ]
        }
    ]
)

print(response.choices[0].message.content)
```

## 注意事项

- 提示词设计要清晰明确，指定需要提取的内容结构
- 处理大批量图片时注意 API 调用频率限制
- 提取结果需要人工审核和校对，确保准确性
- 遵守版权法规，确保有权使用和变现相关内容
